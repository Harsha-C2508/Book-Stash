## Packages
@mantine/core | Core components for the UI
@mantine/hooks | React hooks for state and UI logic
@mantine/dates | Date pickers and calendars
@mantine/notifications | Toast notifications system
dayjs | Date manipulation (dependency for @mantine/dates)
postcss-preset-mantine | PostCSS preset for Mantine
postcss-simple-vars | PostCSS plugin for variables

## Notes
Mantine v7 requires importing CSS files in index.css or App.tsx
Tailwind is available but Mantine components will handle most UI elements
Using Playfair Display for headings and Inter for body text via Google Fonts
